Inheritance 
می‌توان کلاسی را تعریف کرد که برخی از خصوصیات و رفتارهای کلاس‌های دیگر را به ارث ببرد. این کار نگه‌داری و تولید نرم‌افزار شما را بسیار ساده‌تر می‌کند. همچنین باعث می‌شود از توابع استفاده کرده و سرعت برنامه‌نویسی و اجرا پروژه خود را بالا ببرید. اگر کلاس ‌B از کلاس A به ارث ببرد به اصلاح می‌گویند که کلاس B از کلاس A مشتق شده است.


متدهای Extension
با استفاده از متدهای Extension میتوان بدون تغییر و ویرایش کلاس ها و ساختار های از قبل تعریف شده به آنها قابلیت های جدید افزود
یک متد Extension یک متد static بوده که در یک کلاس static تعریف می شود. برای تعریف یک متد Extension ابتدا باید یک کلاس static تعریف کرد.
