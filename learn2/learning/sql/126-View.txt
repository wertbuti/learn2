
نتیجه حاصل از یکسری عبارات و دستورات اس کیو ال بر روی یک یا چند جدول میباشد
برای کاربر مانند یک جدول دیده میشود
ویوها برای مقاصد امنیتی مورد استفاده قرار میگیرند مثلا کاربر خاصی فقط ستون ها و سطرهای خاصی از یک جدول را بتواند دسترسی داشته باشد
-------------------------------------------

انواع View
1) Single Table- این ویو از یک جدول ساخته شده است در این ویو امکان اجرای دستورات DML وجود دارد
2) Multi Tables - این ویو از چند جدول ساخته شده است در این ویو امکان اجرای دستورات DML وجود ندارد
----------------------------------------

دستور ساخت View
Create View MyView 
As
Select * From T1 where Id>1

برای جلوگیری از این کار در هنگام ساخت ویو دستور زیر را مینویسیم
