Select .... Over(order By ... Rows Between X and Y)
X و Y میتواند مقادیر زیر داشته باشد
Unbounded Preceding - تمام سطرهای قبل از سطر جاری
Unbounded Following - تمام سطرهای بعد از سطر جاری
number preceding - بجای numbrer عدد قرار میدهیم - تعدا سطر قبل از سطر جاری
number Following- بجای numbrer عدد قرار میدهیم - تعدا سطر بعد از سطر جاری
Current Row - فقط سطر جاری
