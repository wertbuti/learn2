ساختار فایلهای دیتابیس
به فایلهای دیتابیس dataFile میگویند.
هر دیتا فایل از بلاک های 8 کیلو بایتی به نام page تشکیل شده است.
به هر هشت page کنار هم extend میگویند
ساختار page
header شامل شماره صفحه، نوع صفحه، شناسه صفحه، فضای آزاد موجود در صفحه و مالک صفحه میباشد
data row در این قسمت اطلاعات page ذخیره میشود
free space فضای خالی هر page است
row offset شامل آدرس شروع  data row های page میباشد
