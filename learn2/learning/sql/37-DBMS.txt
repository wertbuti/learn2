DBMS - Database management system
سیستم مدیریت پایگاه داده نرم افزاری است که امکان مدیریت دیتابیس ها را فراهم میکند
-------------------------------------

انواع DBMS
Hierarical سلسله مراتبی - از ساختار درختی برای ذخیره داده ها استفاده میکند
Network از ساختار شبکه ای برای ذخیره داده ها استفاده میکند
Relational از ساختار رابطه ای برای ذخیره داده ها استفاده میکند
NoSQL از ساختار غیرجدولی برای ذخیر داده ها استفاده میکند
