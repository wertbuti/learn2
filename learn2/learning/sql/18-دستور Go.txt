دستور Go در اس کیو ال
کاربرد اصلی آن به عنوان جداکننده  یا خاتمه دهنده دستورات میباشد.مثال
declare @MyVar int
go
print @Myvar
کد بالا خطا میده چون go خاتمه دهنده دستورات است و چون دستور فراخوانی متغیر بعد از go امده است گویا اولین دستور است بنابراین متغییر رانمیشناسد
کاربرد دیگر دستور go استفاده به عنوان حلقه تکرار است اگر جلوی go عددی قرار بگیرد دستورات قبلی تا رسیدن به go به آن تعداد اجرا میشوند مثال
select * From Table1
Go 3
دستور بالا سه بار اجرا مشود
