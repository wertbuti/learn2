تفاوت sp و function
Function‌ها در هربار استفاده کامپایل می‌شوند لیکن SP‌ها می‌توانند یکبار کامپایل شده، execution plan آنها برای استفاده آتی نگهداری شوند. این مهمترین تفاوت این دو می‌باشد.	
در sp میتوان دستورات DML (insert,update,delete) استفاده کرد ولی در function نمیتوان
در sp میتوان از try ... catch استفاده کرد ولی در function نمیتوان
اجرای sp با EXEC یا EXECUTE اجرا و فراخوانی می شوند ولی اجرای Function با Select انجام می شود.
در sp ها جداول موقت یا temporary table ها مجاز بوده ولی در Function اینطور نیست.
sp میتواند function را فراخوانی کند ولی function نمیتواند sp را صدا بزند
sp میتواند هیچ مقدار، یک مقدار و یا چند مقدار برگردالند ولی function حتما باید یک مقدار برگرداند
در داخل sp می توانیم از transzction استفاده کنیم ولی در function نمیتوانیم
sp میتواند هم پارامتر ورودی و هم پارامتر خروجی (input, output) داشته باشد ولی function فقط پارامتر ورودی میتواند داشته باشد
نمی توانیم از sp در عبارت های select, where, having استفاده کنیم. , ولی از function میتوانیم.
