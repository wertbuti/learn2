فایل استریم در اس کیو ال
با استفاده از فایل استریم یک پوشه از نوع NTFS در کنار دیتابیس ساخته میشود و در این پوشه BLOB - باینری لارج آبجکتها ذخیره و نگهداری میشوند
----------------------------------------------------

دلایل استفاده از فایل استریم
1) حجم لاگ دیتابیس کاهش پیدا میکند
2) هنگام بروز رسانی فیلدهایی که در فایل استریم ذخیره شده اند، چون صرفا ایجاد یا حذف یک فایل مدنظر است بنابراین فرگمنت کمتری ایجاد میشود
3) از بافر پول اسکیو ال کمتر استفاده میشود و این بافر جهت استفاده امور روزمره اس کیو ال در دسترس قرار میگیرد
------------------------------------------------

نحوی استفاده از فایل استریم
ابتدا از قسمت زیر فایل استریم را فعال میکنیم
sql server configuration manager > sql server > prperty
سپس از قسمت زیر فول اکسس را انتخاب میکنیم
sql server management studio > Prperty > advanced > file stream accesslevel
سپس در property دیتابیس در قسمت فایل گروپ یک فایل استریم میسازیم و در تب فایل فایلی میسازیم و به آن فایل استریم پیوند میدهیم
سپس یک جدول میسازیم و فایل استریم را برای آن طبق کد زیر تنظیم میکنیم. فیلد جدول حتما باید varbinary(max) باشد
Create Table t1
(Id uniqidentifier RowGUIDCOL NOT Null Unique, Pic varbinary(max) Filestream null)
