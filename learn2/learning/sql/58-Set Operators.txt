Set Operators یعنی عملگرهای مجموعه ای که شامل Union، intersect و Except می باشد
اولویت Intersect از دو عملر دیگر بیشتر است مثل * که از جمع بیشتر است البته اولویت پرانتز بیشتر میباشد دقیقا مثل ضرب و جمع و...
-----------------------------------------------

در هنگام استفاده از Set Operatorها باید تعداد ستون و دیتا تایپ ستون ها یکسان باشند یا دیتا تایپ ها قابل Convert به هم باشند
-------------------------------------

هنگام استفاده از Set Operator ها بر روی مثلا دو کوئری در هر یک از آن کوئری ها نمیتوان order by استفاده کرد اما  بر روی کل کوئری میشود
---------------------------------------------
