تفاوت Table Variable و Temporary able
Table Variable - بر روی رم ذخیره میشوند. حجم کمتری میشود از آن ساخت و سرعتش بالاست
Temporary Table - بر روی هارد ذخیره میشود حجم زیادی از داده ها را میتوان با آن ساخت و سرعتش پایین تر است



تعریف Table Variable از نوع DML میباشد و در رم ذخیره میگردد
اما تعریف Temporary Table از نوع DDL میباشد و در دیسک ذخیره میگردد
