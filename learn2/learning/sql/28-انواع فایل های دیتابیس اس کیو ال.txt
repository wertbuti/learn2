انواع فایل های دیتابیس اس کیو ال
mdf - main data file در این فایل ساختار دیتابیس و داده های دیتابیس قرار میگیرد
ldf - log data file در این فایل لاگ دیتابیس ذخیره میگردد
ndf - not master data file اگر بخواهیم اطلاعات دیتابیس را در چندین فایل تقسیم کنیم اولین فایل mdf و بقیه فایل ها با پسوند ndf مشخص میشوند
